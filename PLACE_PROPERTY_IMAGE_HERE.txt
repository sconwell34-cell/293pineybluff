# Place the property satellite image here

Save the Google Maps satellite screenshot of the property as:

    property.jpg

The site loads `/assets/property.jpg` as the map base image.

When you eventually replace this with a drone shot, keep the same filename
(`property.jpg`) so the site continues to work without code changes.

You may need to adjust the pin coordinates in `content.yaml` if the new
image is framed differently from the old one. See MAINTENANCE.md for how.
